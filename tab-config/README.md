# Configuration TAB pour ZombieZ

## Installation

1. **Télécharger TAB** : https://github.com/NEZNAMY/TAB/releases
2. **Placer le fichier** `config.yml` dans `plugins/TAB/config.yml`
3. **Placer le fichier** `animations.yml` dans `plugins/TAB/animations.yml`
4. **Redémarrer le serveur** ou utiliser `/tab reload`

## Prérequis

- **TAB** (version 4.x+)
- **PlaceholderAPI** (pour les placeholders %zombiez_xxx%)
- **ZombieZ** (avec l'expansion PAPI intégrée)
